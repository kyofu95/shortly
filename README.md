# Shortly
Shortly is a test rest api url shortener

## Prerequisites
* Python 3.11
* FastAPI
* Postgesql 15

## Setup
> git clone https://github.com/kyofu95/shortly
> cd shortly
> cp .env.sample .env
> Edit your .env file and all the credentials
> docker-compose -f docker-compose.yml up
> check localhost:8000/docs
